Methods

(static) jwtSigner.sign(tokenProfile, secretOrPublicKey, options, callback)

Signs JWT and checks for valid input
Parameters:
Name Type Description
tokenProfile contains the token properties,required, optional and verification claims
secretOrPublicKey is a string or buffer containing either the secret for HMAC algorithms, or the PEM encoded public key for RSA and ECDSA
options consists of other inputs that are not part of the payload, for ex : 'algorithm'
callback is called with the decoded payload if the signature is valid and optional expiration, audience, or issuer are valid. If not, it will be called with the error. When supplied, the function acts asynchronously.
