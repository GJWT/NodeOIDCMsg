src/models/keystoreDependency/keyBundle.js

Represents a set of keys with a common origin.The idea behind the class is that it should be the link between a set of keys and the OIDC client library. It works by on one hand keeping an internal representation of the keys in sync with a specific external representation (the external version representing the correct state of the key) and on the other hand provide an API for accessing the keys. The reason for key sets to change are regular key roll-over or as a result of key compromise.
