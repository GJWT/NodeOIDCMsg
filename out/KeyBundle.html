init

Contains a set of keys that have a common origin. The sources can be several: - A dictionary provided at the initialization, see keys below. - A list of dictionaries provided at initialization - A file containing one of: JWKS, DER encoded key - A URL pointing to a webpages from which an JWKS can be downloaded
