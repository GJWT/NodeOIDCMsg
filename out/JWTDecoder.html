decode(jwtString, secretOrPublicKey, tokenProfile, otherOptions, callback)

Decodes Jwt string after verifying if payload matches expected claim values
Parameters:
Name Type Description
jwtString string The signed Jwt string
secretOrPublicKey string A string or buffer containing either the secret for HMAC algorithms, or the PEM encoded public key for RSA and ECDSA
tokenProfile Token Contains the token properties, required, optional and verification claims
otherOptions dictionary Other inputs that are not part of the payload, for ex : 'algorithm'
callback function If a callback is supplied, function acts asynchronously. The callback is called with the decoded payload if the signature is valid and optional expiration, audience, or issuer are valid. If not, it will be called with the error.
